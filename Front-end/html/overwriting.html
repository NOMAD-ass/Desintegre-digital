Destruição Lógica

Overwriting (Sobrescrita)

Os dados são sobrescritos diversas vezes com padrões binários, sem danificar o hardware.

O overwriting (sobrescrita) é um método de destruição lógica indicado para dispositivos que ainda serão reaproveitados, doados ou revendidos. Todo o conteúdo do HD ou SSD é sobrescrito diversas vezes com padrões de dados (zeros, uns e sequências aleatórias), seguindo protocolos reconhecidos internacionalmente, como o NIST 800-88. Ao final, os dados originais não podem mais ser recuperados por nenhum software de recuperação, mesmo os mais avançados - mas o dispositivo continua funcionando normalmente.

Como funciona

1

Diagnóstico do dispositivo

Verificamos o tipo de mídia (HD, SSD ou híbrido) para escolher o algoritmo de sobrescrita mais adequado.

2

Sobrescrita em múltiplas passadas

O software grava padrões de dados sobre 100% dos setores do disco, em várias passadas sucessivas.

3

Verificação de integridade

Uma amostragem dos setores é lida de volta para confirmar que a sobrescrita foi concluída com sucesso.

4

Emissão do certificado

É gerado um relatório com o padrão utilizado, número de passadas e a confirmação de conclusão.

Por que é seguro

Padrão internacional

Seguimos diretrizes como o NIST 800-88, referência mundial em sanitização de dados.

Preserva o hardware

O dispositivo sai funcional, pronto para reuso, doação ou revenda.

Verificação real

Não confiamos só no processo - conferimos amostras dos setores após a sobrescrita.

Precisa deste serviço?

Solicite a coleta e receba um certificado de destruição ao final do processo.
